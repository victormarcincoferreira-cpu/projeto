# Projeto Sprint 5 - Dashboard US Vehicles

Este projeto faz parte do 5º Sprint do curso TripleTen.

O objetivo é analisar e explorar um dataset de anúncios de venda de veículos nos Estados Unidos, criando um dashboard interativo em Python usando **Streamlit**, **Pandas** e **Plotly Express**.

## Funcionalidades do Aplicativo

- Filtros interativos (ano / preço / odômetro)
- KPIs dinâmicos (média de preço, média de quilometragem, ano médio)
- Visualizações interativas:
  - Scatter plot (Preço x Quilometragem)
  - Histograma de preço
- Download do CSV filtrado
